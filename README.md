<h3>To this (smaller to bigger) :</h3>

```css
   width: 100%;
   resize: none;
   padding: 15px;
   outline: none;
   display: block;
   color: #f5f5f5;
   font-size: 18px;
   margin-top: 17px;

   box-sizing: border-box;

```
